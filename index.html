CSS Selectors Cheat Sheet

Selector Description
p Applies the style to all paragraphs (<p> tag).
* Applies the style to all elements on the page.
.classname Applies the style to elements with the class name "classname".
#idname Applies the style to the element with the ID "idname".
div p Applies the style to all <p> elements that are descendants of a <div>.
article > p Applies the style to all <p> elements that are direct children of an <article>.
[href] Applies the style to all elements with an href attribute.
p:first-child Applies the style to the first child of all <p> elements.
p::first-line Applies the style to the first line of all <p> elements.
h1 + p Applies the style to all <p> elements immediately following an <h1>.
